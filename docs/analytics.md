# Ink Ecosystem Analytics

## Overview
CatOnInkChain provides analytics for cross-chain activity related to Ink.

## Metrics
- Weekly netflow (USD)
- Bridge volume
- Source chains
- Inflow vs outflow trends

## Use Cases
- Builder insights
- Ecosystem monitoring
- Market trend analysis

## Example Metrics
- Netflow: Inflow - Outflow
- Volume: Total value bridged

## Future Potential
- AI-driven insights (e.g., InkClaw)
- Builder dashboards
- Automated reports

## Positioning
CatOnInkChain acts as an ecosystem intelligence layer for Ink.
