# Integration Guide for dApps

## Overview
CatOnInkChain can be used as a liquidity onboarding solution for Ink dApps.

## Basic Flow
1. User visits dApp
2. User needs assets on Ink
3. dApp provides bridge button
4. Redirect to:
   https://catonink.com

5. User bridges assets
6. Returns to dApp

## Recommended UI
Button text:
"Bridge assets to Ink"

Placement:
- Near wallet connect
- Near swap or deposit flow

## Benefits
- No need to integrate multiple bridges
- Faster user onboarding
- Access to aggregated liquidity

## Notes
All routing and execution handled by CatOnInkChain.
