# Overview

## What is CatOnInkChain?

CatOnInkChain is a cross-chain bridge aggregator built for the Ink ecosystem.

It allows users to move assets into Ink by automatically selecting the best route across multiple bridge infrastructures.

## Core Functions
- Aggregate multiple bridge providers
- Optimize routes based on liquidity and cost
- Provide a unified interface for bridging into Ink
- Track ecosystem-level liquidity flows

## Supported Infrastructure
- Socket (Bungee)
- Stargate Finance
- Across Protocol
- Relay Protocol

## Not a Bridge Protocol
CatOnInkChain does not operate its own bridge contracts.

Instead, it leverages existing infrastructure and focuses on:
- Routing
- UX
- Ecosystem insights

## Role in Ink Ecosystem
- Entry point for liquidity
- Simplifies onboarding for new users
- Provides insights for builders
