# CatOnInkChain Docs

CatOnInkChain is a cross-chain bridge aggregator and ecosystem analytics platform for Ink.

## Overview
CatOnInkChain helps users and builders:
- Bridge assets into Ink using aggregated routes
- Discover liquidity sources across chains
- Understand ecosystem activity through analytics

## Key Features
- Multi-bridge aggregation (Bungee, Stargate, Across, Relay)
- Smart routing based on cost, speed, and liquidity
- Cross-chain liquidity insights
- Builder-friendly onboarding flows
